# 🎓 Student Placement Portal

## 📌 Overview

Student Placement Portal is a Flask-based web application that helps students manage internship and placement opportunities. Students can register, log in, upload resumes, apply for jobs, and track application status, while administrators can manage applications through an admin dashboard.

---

## 🚀 Features

### 👨‍🎓 Student Features

* User Registration
* User Login Authentication
* Resume Upload
* Internship & Job Listings
* Apply for Jobs
* View Applied Jobs
* Track Application Status
* Logout Functionality

### 👨‍💼 Admin Features

* Admin Dashboard
* View Registered Students
* View Job Applications
* Approve Applications
* Reject Applications
* Update Application Status

### 💾 Database Features

* SQLite Database Integration
* User Data Management
* Application Tracking System
* Status Management (Pending / Accepted / Rejected)
